## 2. Weather Dataset (NOAA / Open-Meteo)

* **Source:** NOAA
* **Alternative API:** Open-Meteo

👉 **Direct API (Columbus, Ohio):**

```
https://archive-api.open-meteo.com/v1/archive?latitude=39.96&longitude=-83.00&start_date=2004-01-01&end_date=2018-08-01&hourly=temperature_2m,relativehumidity_2m
```

✔ **File used:**
`noaa_weather_columbus.csv`

✔ **Description:**

* Location: Columbus, Ohio, USA
* Time range: 2004 – 2018
* Frequency: Hourly
* Features:
  * Temperature (°C)
  * Relative Humidity (%)
